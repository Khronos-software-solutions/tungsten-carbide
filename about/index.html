Welcome to Tungsten Carbide!

Tungsten Carbide is a forum website for questions on subjects like hardware and software. If you are new here, please read the rules before you post something.

Community Guidelines

To make sure we keep the community a friendly place we have some guidelines:
  1. Please refrain from using slurs, continued usage of slurs we result in a permanent ban.
  2. We are not Stack Overflow, nor are we Arch users; keep responses friendly, if someone does not know basic knowledge about a certain subject, please refer them to the appropriate documentation page.
  3. Give correct information; of course, people make mistakes, but giving a blatantly incorrect answer will result in your post getting removed.
  4. If you are not sure, contact one of the moderators, refer to the contact page for more information.
