How to post on the forum

In order to make a post on this forum, you will have to send an email to the moderation team.
This is for moderation purposes and due to the fact that we cannot implement a database yet.
Make sure you read the following before posting

You can submit a post by email: main email (16176@kalsbeek.nl)
alternative email (16182@kalsbeek.nl) We recommend you to send the post request to both addresses.

If you submit something, make sure it contains the following:

When posting:

  • Title
  • Post body (text)
  • Tags (optional, these may be added by our staff)

Or when reacting to a post or answering a question:

  • Post id (can be found in the upper right corner of a post)
  • Reaction body (text)
